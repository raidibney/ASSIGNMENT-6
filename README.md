# 🚀 DigiTools - Premium Digital Marketplace
 A sleek, high-performance digital product storefront built with React and Tailwind CSS. This application allows users to browse a curated collection of premium tools, manage a dynamic shopping cart, and experience seamless UI transitions.
# 📖 Description
 DigiTools is a modern SaaS-inspired marketplace designed for speed and clarity. It features a responsive landing page with a specialized "Premium Tools" section where data is dynamically loaded from a JSON configuration. Users can toggle between the product gallery and their active cart, providing a smooth, single-page application experience.
# 🛠️ Technologies Used
  Frontend Framework: React.js (Vite)

Styling: Tailwind CSS & DaisyUI

Notifications: React-Toastify

Data Handling: Dynamic JSON & Fetch API

Icons & Assets: Custom UI assets and Emoji integration

# ✨ Key Features
 1. Dynamic Product Generation 📦
The "Premium Tools" section automatically generates high-quality product cards from a central products.json file. Each card features:

Unique icons and "Tag" types (Popular, Best Seller, etc.).

Detailed feature lists and pricing periods (Monthly/One-time).

Responsive 3-column layout that adapts to any screen size.

2. Interactive Cart & State Management 🛒
Built with advanced React state lifting, the application features a synchronized shopping experience:

Live Navbar Count: The cart icon in the Navbar updates instantly as items are added.

Smart Toggling: Users can switch between "Products" and "Cart" views without page reloads.

Automatic Totaling: The cart calculates the sum of all selected items in real-time.

3. Professional User Feedback 🔔
Integrated with React-Toastify to provide instant visual confirmation for every user action:

Success Alerts: When an item is added or checkout is completed.

Error Alerts: When items are removed from the cart.

Informational Toasts: Preventing duplicate items from being added.
